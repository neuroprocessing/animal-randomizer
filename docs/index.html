Animal Randomizer

Animal Randomizer is a lightweight open-source tool for generating reproducible, transparent, and publication-ready randomization tables for animal experiments (rats/mice). Designed for rodent neuroscience workflows, but applicable to any biomedical research.

Scientific Motivation Why it matters
  • Bias reduction: prevents uncontrolled allocation.
  • Seed reproducibility: same seed → same random table.
  • Transparent outputs: suitable for supplementary materials.
  • Lab-friendly: minimal dependencies, fast CLI workflow.
  • Extensible roadmap: block and stratified randomization.

Overview

Randomization is a core requirement for rigorous experimental design in animal studies. Animal Randomizer provides a reproducible allocation engine for neuroscience experiments using rodents (rats/mice), producing transparent CSV tables that can be archived, shared, and directly included in publications.

Core Features

The current release supports balanced random assignment across experimental groups, optional custom group names, CSV export, and seed-based reproducibility. All outputs are generated in a clean, auditable format suitable for lab notebooks and reporting.

Installation

git clone https://github.com/neuroprocessing/animal-randomizer.git
cd animal-randomizer
pip install -r requirements.txt

Future releases will support direct installation via pip.

Quick Start

python -m animal_randomizer.cli --n 24 --groups 2 --seed 42 --out output.csv
            
python -m animal_randomizer.cli --n 30 --groups 3 \
  --names Control,DrugA,DrugB \
  --seed 123 \
  --out study.csv

Output files include animal IDs, group labels, random seed, timestamp, and method metadata. This ensures the randomization procedure remains reproducible and fully traceable.

Roadmap

Planned releases include block randomization, stratified randomization (weight/age/baseline behavior), GUI (PyQt6), Excel export, and PDF reporting tools for ethics submissions and manuscript supplements.
