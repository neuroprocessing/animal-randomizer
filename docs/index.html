Neuroprocessing Randomizer

A reproducible randomization and group allocation system for laboratory animal studies, designed for scientific rigor, auditability, and practical lab workflows.

Current: v1.0.0 (Release) CLI + PyQt6 Wizard GUI Welcome Window + Help System Auto Animal List Generator .nprj Project Format SHA256 Integrity Hashing HTML Report Export Windows .exe Build

Implemented Scientific Features

Randomization Strategies

simple, balanced, stratified, and block randomization with fixed or random block sizes.

Bias Controls

Cage-aware constraints, weight-bin balancing, and stratification by sex, cage, weight, and age.

Validation Stats

Group N, weight mean/SD, sex/cage distributions, Cohen's d effect size, and imbalance warnings.

Reproducibility

User-defined or auto seed, deterministic hashes for input/config/output, and JSON audit trail.

Outputs

Allocation export to .csv/.xlsx/.tsv, Prism-grouped tables, HTML report, and full .nprj snapshot.

Modular Architecture

Separation of core logic, I/O, reporting, and UI to support future integration in Animal Study Manager.

Wizard GUI (Implemented)

PyQt6 interface with a guided 5-step workflow for non-programmers:

  • Create Study
  • Import Animals
  • Configure Strategy and Constraints
  • Run Randomization
  • Review and Export

Includes separate Welcome window, in-app Help button, automated animal list generation, dropdown/date input controls, and export summary guidance.

Install

git clone https://github.com/neuroprocessing/animal-randomizer.git
cd animal-randomizer
pip install -e .

CLI Quick Start

animal-randomizer \
  --input examples/example_dataset.csv \
  --study-id NP-2026-001 \
  --title "Cognitive Rescue Pilot" \
  --researcher "Dr. N. Process" \
  --institution "Neuroprocessing Lab" \
  --method stratified \
  --groups Control,DrugA,DrugB \
  --stratify-by sex,cage,weight \
  --random-block-sizes 4,6,8 \
  --max-cage-per-group 1 \
  --seed 20260217 \
  --export-bundle \
  --out-alloc examples/example_allocation.csv \
  --out-report examples/example_report.html \
  --out-project examples/example_project.nprj

GUI launch: animal-randomizer-gui

Windows Release

powershell -ExecutionPolicy Bypass -File .\scripts\build_windows.ps1
