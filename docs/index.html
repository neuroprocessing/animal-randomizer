Reproducible random group assignment for animal studies.

Animal Randomizer is a lightweight open-source tool for generating transparent and seed-based randomization tables for rodent experiments (rats/mice). Built by Neuroprocessing for modern reproducible neuroscience workflows.

Scientific Motivation

  • Bias reduction: controlled random group allocation.
  • Reproducibility: seed-based and audit-friendly outputs.
  • Transparency: ready for supplementary materials.
  • Lab usability: minimal dependencies, fast workflow.

Core Features

Seed-Based Reproducibility

Same seed → same allocation. Fully auditable for papers and replication studies.

Balanced Group Allocation

Automatic distribution ensures balanced sample sizes across groups.

CSV Export

Export publication-ready tables for lab notebooks, Excel, or manuscript supplements.

Custom Group Names

Use meaningful labels like Control, DrugA, DrugB.

CLI Workflow

Designed for real lab use: fast execution and clean outputs.

Roadmap Extensions

Block randomization, stratification (weight/age), GUI, PDF reports.

Installation

git clone https://github.com/neuroprocessing/animal-randomizer.git
cd animal-randomizer
pip install -r requirements.txt

In future versions, Animal Randomizer will support installation via pip install animal-randomizer.

Quick Start

python -m animal_randomizer.cli --n 24 --groups 2 --seed 42 --out output.csv

This generates a reproducible randomization table containing animal IDs, group labels, seed, timestamp, and method metadata. 

python -m animal_randomizer.cli --n 30 --groups 3 \
  --names Control,DrugA,DrugB \
  --seed 123 \
  --out study.csv

Output Format

Animal_ID

Automatically generated IDs such as RAT_001.

Group

Assigned experimental group label.

Seed + Timestamp

Ensures reproducibility and transparent reporting.
